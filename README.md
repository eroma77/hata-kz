# Hata.kz 🏠

Сервис поиска и аренды/продажи недвижимости в Казахстане.

## Стек технологий
- HTML5 (Семантическая структура)
- CSS3 (Современный дизайн, переменные, анимации)
- JavaScript (Ванильный JS для интерактива)

## Запуск проекта
Просто откройте `index.html` в браузере или запустите локальный сервер (например, Live Server).
